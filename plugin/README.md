# SignalMirage

A focused monochrome transmission system for Photoshop.

## v1.5.0 release candidate

This build is the cleaned SignalMirage release-candidate pass.

### Current workflow

- Source: LIVE Preview
- Preset: Lucid, Ghost, Veil, Echo
- Customize: Atkinson or Floyd-Steinberg
- Tone: Exposure Bias and Tonal Response
- Range: Black Point and White Point
- Structure: Dot Size and Diffusion
- Atmosphere: Transmission Clarity

### Preset Range Recipes

- Lucid: Black Point 0 / White Point 255
- Ghost: Black Point 140 / White Point 244
- Veil: Black Point 120 / White Point 250
- Echo: Black Point 60 / White Point 245

### Cleanup

- Removed unused Palette logic.
- Removed unused Signal Spread logic.
- Removed hidden Artifact and Edge fallback logic.
- Removed contamination, grain, dry plate, and toner leak rendering code from this build.
