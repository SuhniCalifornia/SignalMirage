# SignalMirage

A monochrome transmission-based dithering and texture system for Photoshop.

## v1.1.1

This version keeps SignalMirage focused as a two-color monochrome transformation tool. It removes the four-color palette system, keeps Ghost + Dance and Midnight + Moon, refines Dot Size and Grain Scale into subtler 10-step controls, preserves first-preview engine initialization, and simplifies the footer status messages.
