# Changelog

All notable changes to SignalMirage will be documented in this file.

SignalMirage is early, so this changelog keeps released work and planned work in one place.

## [1.0] - Initial Public Release

### Added

- Atkinson dithering
- Floyd-Steinberg dithering
- Threshold dithering
- Printer Drift
- Signal Response
- Transmission Clarity
- Dry Plate
- Toner Leak
- Contamination System
